# CRAV Burgers

URL: https://www.cravburgers.shop/

見た日: 2026-07-14

タグ: loading, scroll, micro-interaction, sticker, transition

## 良かったところ

- 待ち時間を、具材が順番に積み上がる楽しいローディング体験へ変えている。
- シールが単に現れるのではなく、折り返し面を見せながら貼り付く演出に手触りがある。

## 今回抜き出す要素

具材が積み上がるローディングと、スクロールで貼り付くシール演出を、1つのデモサイト内で連続して体験できるようにする。

## 使えそうな場面

- 食品・雑貨など、複数の要素を組み合わせる楽しさを伝えたいブランドサイトの初回表示。
- キャンペーンLPや特設サイトで、スクロールに参加感を持たせながら装飾を完成させたい場面。

## 挙動仕様

### ローディング

- レンダリングモデル: 画面固定のローダー内に6枚の具材SVGを絶対配置し、前面に進捗と状況文、背面に3枚の全画面SVGカーテンを重ねる。
- 直接観測: 公開JSはGSAPタイムライン。具材は `y: -800 / opacity: 0 / scale: .8` から0.22秒間隔で落ち、各0.32秒の `power1.in` で着地する。続けて目標角度±12度から0.45秒の `back.out(2.5)` で回転を戻す。
- 直接観測: 進捗は通信量ではなく、1.5秒の `power1.inOut` で0から100へ進む。各着地では全体を `scaleY: .88 / scaleX: 1.06` まで潰して戻す。
- 直接観測: 約1.42秒で14粒のパーティクル、約1.72秒で完成メッセージとバウンス、約2.62秒から赤・橙・えんじの3枚を0.12秒差で上へ抜く。全体は約3.96秒。
- 直接観測: 実ブラウザでは開始時に6パーツすべてが非表示、0.25秒で進捗20・2層目が着地中、0.8秒で進捗96・最上層が着地中、1.1秒で全層と進捗100、3.4秒時点でローダーDOMが外れることを確認した。
- 不変条件: 「順番に落下 → 着地の潰れ → 完成の祝福 → 曲線カーテンで画面を開く」の順序を保つ。

### シール

- レンダリングモデル: `.sticker-main` と、同じ画像を上下反転した `.flap` を重ね、共通の折り目位置で2枚を切り替える。
- 直接観測: ScrollTriggerは標準で `start: "-120% 50%" / end: "30% 50%" / scrub: 1 / ease: "none"`。CTAの大きなシールだけ `"0% 50%" → "50% 50%"`。CSS変数 `--peel-amount` を1から0へ連続更新し、逆スクロールでは元へ戻る。
- 直接観測: `--sticker-p: 20%`、折り目は `-20% + peelAmount × 140%`。表面は折り目から下、裏面は-20%から折り目までを `clip-path` で表示し、裏面を `top: calc(-100% + 2 * peelY - 1px)` と `scaleY(-1)` で接続する。
- 直接観測: 実ブラウザでは `scrollY: 0 / peel: 1`、`690 / .580618`、`920 / .048594`、`1139.5 / .000004`、`1359.5 / 0` と連続変化し、表面のクリップ境界と裏面位置が同じ値で動くことを確認した。
- 直接観測: 細かいポインターではSVG照明がマウス位置を追い、粗いポインターとSafariでは照明・フィルターを簡略化する。対象演出自体には `prefers-reduced-motion` 分岐がない。
- 推論: 折り返し面が下側から巻き上がり、貼り付いた表面が増える見え方になる。形状計算と実ブラウザの途中状態から判断した。
- 不変条件: 単純なフェードやスタンプ動作へ置き換えず、表面・裏面・折り目を1つのスクロール進捗で同期する。

## 実装メモ

- 構造: 架空ブランド「STACK! DELI」のページを新規制作。ローディングの6層はCSS図形、シールは同一のオリジナルラベルを表面と反転フラップへ2重配置した。
- 動き: ローダーは0.22秒間隔の落下、着地スクワッシュ、パーティクル、3枚のSVGカーテンを約4秒で再構成。カーテンの二次曲線は毎フレーム `d` を更新する。シールは `--peel` と `clip-path` をスクロール位置から連続計算し、逆スクロールにも追従する。
- 余白・サイズ: ローダーは `100dvh`。デモ本体はPCで2カラム、900px以下で縦積み。シール場面はsticky要素を約1.9画面分の区間に置き、貼る途中を見届けられる距離を確保した。
- アクセシビリティ: ローダーに `role="dialog"`、状況文に `aria-live="polite"`、進捗に `role="progressbar"` を設定。操作可能な再生ボタンと明瞭なフォーカス表示を追加した。`prefers-reduced-motion: reduce` ではローダーを省略し、シールを貼り終えた静止状態にする。
- 独自化: 参照サイトの画像・SVGパス・ロゴ・文言・配色は使用していない。公開コードは挙動値の確認だけに使い、HTML/CSS/JSは小さな独自実装として書き直した。
